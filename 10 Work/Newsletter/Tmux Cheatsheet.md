## Session Management
| Command               | Action                      |
| :-------------------- | :-------------------------- |
| `tmux`                | Start new session           |
| `tmux new -s name`    | Start new named session     |
| `tmux ls`             | List sessions               |
| `tmux attach -t name` | Attach to session           |
| `Ctrl+b d`            | Detach from session         |
| `Ctrl+b s`            | List sessions (interactive) |
