Cargo is rusts compiler. Some commands are:
- cargo new
    - creates a new project
- cargo build
    - build the project
- cargo run
    - build and run the project with one command
- cargo check
    - build project with no binary to check if code will compile
- cargo build --release
    - builds a release version of the program. takes longer to compile but comes with optimizations
